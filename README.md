### <div align="center"> 구인 구직 사이트 </div>

- **프로젝트 명 :** 구인 구직 사이트
- **프로젝트 기간 :** 2024.07.12 ~ 2024.07.19

- ## 소개 🔍
- Spirng Legacy를 이용한 구인 구직 사이트입니다.
- DB는 Oracle이용하고 MyBatis를 이용하여 구현했습니다
- 주요기능은 이력서 C/R/U/D, 공고C/R/U/D, 공고 필터링, 스크랩기능을 구현하였습니다.
